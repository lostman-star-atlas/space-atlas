> ⭐ *Space Atlas is an independent tool officially recommended and promoted by DWARFLAB to their global user community.*

## Features
- **Smart Exposure & Gain:** Dynamically calculates optimal settings combining target surface brightness, live Bortle scale, and moon phases.
- **Live Weather & Seeing:** Powered by Open-Meteo API for real-time cloud cover and automatic seeing estimations via GPS.
- **PWA Ready:** No install required, works offline on iOS/Android browsers when added to the home screen.
- **Privacy First:** Zero tracking, no registration, data stored locally with CSV export.
